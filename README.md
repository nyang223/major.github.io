# PNU 학·석사/학·석박사통합 연계과정 홍보 홈페이지

## GitHub Pages 업로드 방법
1. 이 폴더의 `index.html`, `style.css`, `.nojekyll` 파일을 GitHub 저장소에 올립니다.
2. Settings → Pages → Branch를 `main` 또는 `master`로 선택합니다.
3. 배포 주소 예: `https://아이디.github.io/저장소명/`

## 수정해야 할 부분
- `index.html`의 `proposedRule` 부분: 개정 추진 중 규정(안)의 세부 기준이 확정되면 자동 판정 로직에 반영하세요.
- `어떤 학과가 있는가?` 섹션: 대학원 학과별 소개, 연구실 지원, 연락처, 홍보 링크를 카드로 추가하세요.
- 접수 일정은 2026학년도 1학기 모집요강 기준입니다. 다음 모집 시 날짜만 교체하면 됩니다.
